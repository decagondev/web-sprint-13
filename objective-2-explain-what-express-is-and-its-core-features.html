Understanding Express and Its Core Features

What is Express?

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications. It sits on top of Node.js's built-in HTTP module, simplifying its APIs and adding helpful new features.

Think of Express as a layer built on top of Node.js that helps manage servers and routes. It's like a toolset that makes it much easier to build web applications.

What Can You Build with Express?

Web Applications

Full-featured web applications with server-side rendering

RESTful APIs

JSON-based web services and APIs

Real-time Apps

WebSocket and WebRTC based applications

Core Features

Middleware

Functions that have access to the request and response objects. They can:

  • Execute any code
  • Modify request/response objects
  • End request-response cycle
  • Call next middleware

Routing

Table of contents for your application, determining how it responds to client requests to particular endpoints (URIs) and methods.

Static File Serving

Built-in middleware for serving static files like images, CSS, and JavaScript.

app.use(express.static('public'));

Template Engines

Support for multiple template engines to generate HTML dynamically.

Benefits

  • Simple and unopinionated
  • Extensible with middleware
  • Large ecosystem of plugins
  • Easy to learn and use

Drawbacks

  • Not a full-featured framework out of the box
  • Requires more decision-making due to flexibility

Your First Express Server

Setup Steps

  1. Create a new file: server.js
  2. Initialize npm: npm init -y
  3. Install Express: npm install express
const express = require('express'); // import the express package

const server = express(); // creates the server

// handle requests to the root of the api, the / route
server.get('/', (req, res) => {
    res.send('Hello from Express');
});

// watch for connections on port 9000
server.listen(9000, () =>
    console.log('Server running on http://localhost:9000')
);

Running Your Server

Start the server: node server.js
Visit: http://localhost:9000
Stop the server: Press Ctrl + C
