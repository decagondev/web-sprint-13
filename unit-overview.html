Unit Overview

Web Unit 4 Sprint 13: Node.js and Express

Introduction Video

Learning Objectives

Technical Skills

  • Build RESTful web services using Node.js and Express
  • Implement server-side routing and middleware
  • Handle HTTP requests and manage server responses
  • Create and organize API endpoints effectively
  • Deploy web services to production environments

Professional Skills

  • Debug and troubleshoot server-side applications
  • Write clean, maintainable backend code
  • Document API endpoints and their usage
  • Implement security best practices
  • Work with environment variables and configuration

Unit Structure

Module 1: Introduction to Node.js and Express

Foundation of server-side JavaScript and Express framework basics.

Key topics: Node.js runtime, npm, Express setup, basic routing

Module 2: Server-Side Routing

Deep dive into Express routing and REST architecture.

Key topics: Route parameters, query strings, REST principles

Module 3: Express Middleware

Understanding and implementing Express middleware.

Key topics: Custom middleware, error handling, authentication

Module 4: Deployment and Best Practices

Preparing and deploying web services to production.

Key topics: Environment configuration, deployment, testing

Assessment Information

This unit will be assessed through a combination of:

  • Daily guided projects and exercises
  • Module project implementations
  • Sprint challenge assessment
  • Code quality and best practices evaluation
