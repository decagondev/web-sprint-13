Key Concepts

Express Router

Learn how to use Express Router to modularize your application's routes and create a scalable structure.

Route Parameters

Handle dynamic values in URLs and access them through req.params.

Query Strings

Process URL query parameters and access them through req.query.

Request Body

Handle data sent in POST/PUT requests through req.body.

Code Examples

Express Router Setup

// userRouter.js
const express = require('express');
const router = express.Router();

router.get('/', (req, res) => {
    res.json({ message: 'Get all users' });
});

router.get('/:id', (req, res) => {
    const { id } = req.params;
    res.json({ message: `Get user ${id}` });
});

module.exports = router;

// server.js
server.use('/api/users', userRouter);

Handling Query Parameters

router.get('/search', (req, res) => {
    const { sort, filter } = req.query;
    res.json({ 
        message: 'Search results',
        sort,
        filter
    });
});
