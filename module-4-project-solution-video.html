Module 4 Project Solution

Deployment and Best Practices Implementation

Important Note About Deployment Options

While the video demonstrates deploying to Heroku, which has traditionally been a popular choice, as of November 2022 Heroku does not provide a totally free tier. We've included both the Heroku solution and an alternative solution using Render.com, which offers a generous free tier perfect for personal projects. You can choose the platform that best suits your needs.

Solution Walkthrough

Deployment Solutions

Key Implementation Steps (Heroku)

  1. Project Configuration:
    • Update package.json with start script
    • Configure dynamic port assignment
    • Set up environment variables
  2. Database Setup:
    • Configure database connection string
    • Handle database migrations
    • Set up database seeds
  3. Deployment Process:
    • Create Heroku application
    • Set up Heroku CLI
    • Configure environment variables
    • Deploy application

Note: The video demonstrates these steps in detail. While Heroku now requires a credit card for account verification, the deployment process remains the same.

Key Implementation Steps (Render.com)

  1. Project Configuration:
    • Update package.json with start script
    • Configure dynamic port assignment
    • Set up environment variables
    • Push code to GitHub repository
  2. Database Setup:
    • Create a PostgreSQL database in Render.com
    • Configure database connection using Internal Database URL
    • Set up database migrations and seeds
  3. Deployment Process:
    • Create new Web Service in Render.com
    • Connect GitHub repository
    • Configure build and start commands
    • Set environment variables

Pro Tip: Render.com provides a more streamlined deployment process with automatic HTTPS, continuous deployments from GitHub, and a user-friendly dashboard for managing your services.

Database Note: Render.com offers a free PostgreSQL database with 1GB storage, perfect for development and small projects. The database connection is automatically secured and can be easily integrated with your web service.
