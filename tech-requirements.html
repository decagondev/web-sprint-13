Detailed Requirements

Node.js

Version 18.x or later is required for:

  • Modern JavaScript features
  • Enhanced performance
  • Better security features
  • Long-term support (LTS)
node --version

npm (Node Package Manager)

Version 9.x or later provides:

  • Improved dependency resolution
  • Better security features
  • Faster package installation
  • Enhanced error reporting
npm --version

Visual Studio Code

Recommended extensions:

  • ESLint
  • Prettier
  • Node.js Extension Pack
  • REST Client

Configure auto-formatting on save for consistent code style

Postman

Essential features for API testing:

  • Request collections
  • Environment variables
  • Automated testing
  • Request history

Create a workspace for this sprint to organize your API tests

HTTPie

Command-line HTTP client with intuitive syntax

http POST :3000/api/users name=John

SQLite Studio

GUI for SQLite database management

  • Visual query builder
  • Data import/export
  • Schema management

Nodemon

Auto-restart Node.js applications

npm install -g nodemon

System Requirements

Hardware

  • 4GB RAM minimum (8GB recommended)
  • 2GHz dual-core processor or better
  • 10GB free disk space
  • Stable internet connection

Operating System

  • Windows 10/11
  • macOS 10.15 or later
  • Linux (Ubuntu 20.04 or similar)

Common Setup Issues

Known Issues and Solutions

  • Node.js version conflicts: Use nvm (Node Version Manager)
  • npm permissions: Configure npm to use a different directory
  • Port conflicts: Check for running processes on port 3000
  • Git authentication: Set up SSH keys or use credential manager
